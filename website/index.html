How it thinks

A deliberate agent loop

01
Understand context
Reads your message, loads workspace memories, checks active workspace path and framework.
02
Emit a plan
For complex tasks, shows a step-by-step plan before acting so you always know what's coming next.
03
Execute with real tools
Commands run in your actual shell. File writes happen on your real filesystem. Results stream live.
04
Self-correct
If a command fails, the agent reads the error, diagnoses it, and retries — up to 10 iterations.
05
Learn and remember
After responding, auto-extracts facts to memory so every next session is smarter.
